L2Tool
======
Lineage 2 texture viewer/replacer.

![L2Tool Screenshot](images/screenshot.png)

Run
---
```bash
# Build the project
./gradlew build

# Run with JavaFX (extracts javafx-17.0.2.zip automatically)
./run.bat
```

Build
-----
```bash
./gradlew build
```

Requirements
------------
Java 17 or later is required.
